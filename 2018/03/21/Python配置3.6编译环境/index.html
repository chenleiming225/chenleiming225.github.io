Python配置3.6编译环境

2018-03-21

Python

PHP 是世界上最好的语言,但，人生苦短，我用 python

1. Python 下载

  • 下载路径:https://www.python.org

2. Python 添加路径

  • 安装 Python 不再敖述，一直 next 即可，保留安装路径，配置时要用到。

  • 依次打开“控制面板—>系统和安全—>系统—>高级系统设置“，在打开的面板中依次选择“高级—>环境变量”

    在环境变量配置面板中将已安装的 Python 的路径编辑进去

  • 检查是否路径配置成功
    任意位置依次按下”Win+R”—>”cmd”打开命令行，在命令行内输入“Python”出现版本号即成功，在命令行窗口内输入print(“Hello World! Hello Python!”) 如下图输出即为成功！

3. Python 2.7编译更换为3.6

打开已下载好的 PyCharm 在主菜单栏依次“view—>Toolbar—>settings—>Project Interpreter” 在编译器中配置编译器路径，将 Python 2.7 更换为 Python 3.6
Python 3.6 与 Python 2.7 相比，支持 Unicode 中文变量输入。
